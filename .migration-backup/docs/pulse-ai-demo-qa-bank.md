# Pulse AI Demo Q&A Bank

Use this as the scripted demo knowledge base for the floating Pulse assistant.

Source of truth:

```text
demo-data/pulse-ai-demo-dialogues.json
```

Implementation rules:
- Show suggested questions depending on the current route.
- Clicked question displays the matching scripted answer.
- Free text can show the closest scripted answer or a safe fallback.
- No real AI API in first demo.
- EN/AR supported.
